# Hotel Management System

A C-based hotel reservation and guest management system designed to digitize guest records, reservation management, and guest tracking.

## Features

- Employee login authentication
- Add, view, update, delete, and search reservations
- Manage guest information
- Guest check-in and check-out tracking
- View currently occupied rooms
- Store and retrieve records using text files

## Language

- C Programming Language

## File Handling

The system uses text files to store and manage data:

- `GuestInformation.txt` - Stores reservation and guest details
- `CheckInInformation.txt` - Stores guest check-in records
- `CheckOutInformation.txt` - Stores guest check-out records

Originally created as my CAPE Computer Science Unit 1 IA and rebuilt to strengthen my C programming skills, improve software design practices, and gain more experience with data structures and file handling.
